# Use cases

## See the time
I don't show the clock on my operating system to basically save space on the window. Because of that I want an easy way to see the current time.

This scenario is already covered as I can press SHIFT+CTRL+ALT+C and toggle the clock

## See UTC
Sometimes I am comparing dates with different system and when I am looking at a date I want to understand what is the UTC time. I want to visualise the UTC time along side the current time. 

## See Other timezones
I want to know what time is in other timezone.
This view should display
the current time and date
tue utc time and date
the timezone time and date
